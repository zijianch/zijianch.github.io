Software

The followings are some of the software of which I am one of the developers. Other useful software can be found in Misc>>Resources.

Brain Networks Analysis

Hodge Lapalacian

The code performs the Hodge Laplacian based modeling of cycles and various Hodge Laplacian based computations in brain networks. Click here for details of the algorithm and the usage.

The source codes can also be found at this GitHub repo. If you are using any of the codes given here, please reference the paper

Dakurah, S., Anand, D.V., Chen, Z., Chung, M.K. 2022, Modeling Cycles in Brain Networks Using Hodge Laplacian, Medical Image Computing and Computer Assisted Intervention - MICCAI 2022. Lecture Notes in Computer Science, vol 13431. Springer, Cham.

Spherical Multi-dimensional Scaling

The codes performs the spherical- and hyperbolic-MDS.

The source codes can also be found at this GitHub repo. If you are using any of the codes given here, please reference the paper

Chung, M.K., Chen Z. 2022, Embedding of Functional Human Brain Networks on a Sphere. https://arxiv.org/abs/2204.03653

Computational Neuroimaging/Neuroanatomy

Heat Kernel Smoothing on circular data

This function provides a smooth estimate of the functional data (corpus callosum boundaries) defined on a circle. This is for matching two circular functional data for cross comparison that reduces spatial variability in statistical analysis.

The source codes can be found at this GitHub repo.
